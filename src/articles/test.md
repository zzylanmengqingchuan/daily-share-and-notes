这是最近几天看到的一款非常不错的产品，是清华大学开源的一款名字叫做 OpenMAIC 的产品。

它以网页的形式存在，主要功能是为了进行学习，类似于 NotebookLM 的那样一种功能。但它是免费的，而且我觉得效果在某些方面比 NotebookLM 还要好。

首先，你只需要在浏览器搜索 OpenMAIC 就可以进入使用了。下面是具体的工作台页面情况：
![image.png](https://zzy-1326340203.cos.ap-beijing.myqcloud.com/20260319183704476.png)
它生成的课程形式非常丰富，包含了通过视频介绍的具体的 PPT，还包含了一些交互式的动画、讨论以及课堂测验。总之形式多种多样，我觉得非常值得去体验。

下面是我使用这款工具去做的一个和《金瓯缺》相关的讲解，效果还是非常不错的。

目前这个产品是免费的，每天有 10 次使用的额度，也就是可以生成 10 份课程。因为在生成课程的过程中，消耗的 Token 肯定是比较大的，它包含了视频、音频、PPT，还包含了一些课堂测验以及交互式设计，这些普通的模型肯定是做不了的。

而且他们的代码也是开源的，我把代码下载下来并填入自己的 API Key 尝试过，发现普通的模型 API 确实达不到这个效果，必须需要一些支持多模态的模型。

考虑到生成这种效果的 Token 消耗量以及开销都不小，我建议大家还是直接去官方平台使用免费额度。每天 10 次的使用机会，我觉得基本上是足够的。
